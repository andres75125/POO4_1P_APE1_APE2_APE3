# POO4_1P_APE1_APE2_APE3
Proyecto 1P: Avance 2
# ⚽ Sistema de Venta y Gestión de Entradas - Mundial 2026

Este repositorio contiene el desarrollo del proyecto práctico del Primer Parcial para la materia de **Programación Orientada a Objetos (POO)** en la **Escuela Superior Politécnica del Litoral (ESPOL)**. La aplicación consiste en un sistema interactivo de consola desarrollado en Java enfocado en aplicar de forma estricta los principios de POO.

## 🏗️ Estructura del Proyecto

De acuerdo con los requerimientos arquitectónicos, el código fuente está organizado dentro del directorio `src`:

📂 mundial-poo-project
 ┣ 📂 src
 ┃ ┣ 📄 Main.java       # Flujo principal, menús de interacción y login por consola
 ┃ ┣ 📄 Cliente.java    # Clase modelo para usuarios/aficionados (Aplica Herencia)
 ┃ ┣ 📄 Compra.java     # Clase modelo encargada del registro de transacciones
 ┃ ┗ 📄 Partido.java    # Clase modelo con la información y disponibilidad de los encuentros
 ┣ 📄 .gitignore        # Archivo de exclusión de Git para evitar conflictos
 ┗ 📄 README.md         # Documentación general del repositorio

## 🛠️ Tecnologías y Entorno
- **Lenguaje:** Java JDK 17 o superior
- **IDE Recomendado:** Visual Studio Code (con *Extension Pack for Java*)
- **Control de Versiones:** Git y GitHub
